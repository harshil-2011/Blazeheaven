Connection Guide

Java Edition

  1. Open Minecraft and click "Multiplayer"
  2. Click "Add Server"
  3. Enter server details and click "Done"
  4. Select the server and click "Join Server"

Mobile/Console

  1. Go to "Servers" tab
  2. Scroll down and click "Add Server"
  3. Enter server details
  4. Save and connect

Windows 10

  1. Click "Play" then "Servers"
  2. Click "Add Server" at the bottom
  3. Enter server details
  4. Save and connect
